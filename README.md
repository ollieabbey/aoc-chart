# aoc-chart
Page displaying Advent of Code stats and charts, written using Next.js.

## Prerequisites
- `node`
- `yarn`
- An Advent of Code session token.

## Setup

To setup the project and download the dependencies, change directory into the project root and run
```shell
$ yarn
```

Export your Advent of Code token as an environment variable
```shell
$ export AOC_SESSION=Paste your token here
```

Start up with `yarn`
```shell
$ yarn dev
```
